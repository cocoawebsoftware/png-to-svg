# PNG → SVG ベクター変換 Webアプリ

## 概要
このWebアプリは、PNG画像をSVGに変換し、色数・細かさ・滑らかさ・ノイズ除去・色簡略化などの高度設定を自由に調整できるツールです。  
ドラッグ＆ドロップで画像を読み込み、プリセットやカラー/白黒切替も可能。変換後はBefore/After比較ができ、SVGは最適化してダウンロードできます。

---

## 特徴
- ドラッグ＆ドロップで簡単アップロード  
- プリセット：ロゴ / 写真 / アイコン / カスタム  
  - 高度設定変更時は自動で「カスタム」に切替  
- カラー/白黒切替ボタン（高度設定とは独立）  
- 高度設定スライダー：
  - 色数（数値表示）
  - 細かさ
  - 滑らかさ
  - ノイズ除去
  - 色簡略化
  - infoアイコンで各項目の説明表示  
- 最適化チェック（軽量化）  
- Before/After比較 + 切替ボタン  
- ダウンロードボタン（画像＋文字）  
- UI中央寄せ、レスポンシブ対応  
- 日本語部分はシステムフォント、英語はGill Sans Light / Italic使用

---

## 使い方
1. `index.html` をブラウザで開く  
2. PNG画像をドラッグ＆ドロップ、またはクリックで選択  
3. プリセットや高度設定を調整  
4. カラー/白黒モードを切替  
5. 「変換」ボタンでSVG生成  
6. Before/Afterで比較  
7. ダウンロードボタンでSVGを保存
