# PNG to SVG Vector Converter Web App
s
## Overview
This web app converts PNG images into SVG vector graphics with customizable settings.  
Users can adjust color count, detail level, smoothness, noise removal, and color simplification.  
Drag & drop images, apply presets, switch between color/black-and-white modes, and compare Before/After results. The generated SVGs can be optimized and downloaded.

---

## Features
- Drag & drop or click to upload PNG images  
- Presets: Logo / Photo / Icon / Custom  
  - Automatically switches to "Custom" when advanced settings are modified  
- Color / Black & White toggle buttons (independent of advanced settings)  
- Advanced settings sliders:
  - Number of colors (with numeric display)
  - Detail level
  - Smoothness
  - Noise removal
  - Color simplification
  - Info icons provide explanations for each setting  
- Optimization checkbox for lighter SVG output  
- Before/After comparison with toggle buttons  
- Download button with image + text  
- Centered UI, responsive layout  
- Japanese text uses system font, English text uses Gill Sans Light / Italic

---

## Usage
1. Open `index.html` in a browser  
2. Drag & drop or click to select a PNG image  
3. Adjust presets and advanced settings as desired  
4. Toggle Color/Black & White mode  
5. Click "Convert" to generate SVG  
6. Compare Before/After results  
7. Click the download button to save the SVG
